Install Playwright Browsers

cd frontend
pnpm exec playwright install

This will download Chromium, Firefox, and WebKit (about 500MB total). It takes a few minutes.

After Installation, Run Tests Again

# From project root

./dev.sh test:e2e

Or if you want to test just one browser to start:

cd frontend
pnpm exec playwright test --project=chromium
